値を特定の集合に制限したい場合があります。たとえば、ある値が 1、2、3 のいずれかであることを保証したい場合です。

**🔢 Limiting Value Gate:**

$$
s \times (a - 1)(a - 2)(a - 3) = 0
$$

- この gate は、$s = 1$ のとき $a$ が 1、2、3 のいずれかの値しか取れないことを強制します。
- $a = 1$ の場合、最初の項 $(a - 1)$ が 0 になり、式全体が 0 になります。
- $a = 2$ の場合、2 番目の項 $(a - 2)$ が 0 になり、式全体が 0 になります。
- $a = 4$ や $a = 99$ の場合、どの項も 0 にならないため、式は 0 になりません。
- 同様に、$a$ が他の値の場合、この gate は $s = 0$、つまり selector が無効でない限り equation を満たしません。
